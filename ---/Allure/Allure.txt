Allure

Установка:
1. pip install allure-pytest  — Python-плагин для pytest, который собирает сырые данные о тестах в папку allure-results (JSON-файлы)
2. brew install allure        — CLI-инструмент (на Java), который читает эти данные и генерирует красивый HTML-отчёт / запускает веб-сервер для просмотра

-------------------- allure-results -------------------
python -m pytest -s -v --alluredir=./allure-results --clean-alluredir     - запуск ВСЕХ тестов и сохранение отчета Allure в папке <allure-results> (+отчистка)

allure serve allure-results                                               - открытие отчета в браузере

-------------------- allure-report --------------------
allure generate allure-results -o allure-report --clean   - Сгенерировать отчет из allure-results в папку allure-report (+отчистка)
allure open allure-report                                 - Открыть уже сгенерированный отчет:

_______________________________________
Обновление:
brew upgrade allure — Обновление
allure --version — Проверка версии
